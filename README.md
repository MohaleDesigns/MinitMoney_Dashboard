# MiniMoney Dashbord - Financial Management Dashboard

A modern, responsive financial management dashboard built with React, TypeScript, and Vite. MiniMoney provides a comprehensive interface for managing financial transactions, and user accounts.

## 🚀 Features

### Core Functionality
- **Dashboard Overview**: Real-time financial metrics and performance indicators
- **Transaction Management**: Complete transaction history with status tracking
- **User Management**: User account administration and role-based access control
- **Responsive Design**: Mobile-first approach with adaptive layouts

## 🏗️ Architecture

### Frontend Stack
- **React**: Latest React with modern hooks and patterns
- **TypeScript**: Type-safe development with strict type checking
- **Vite**: Fast build tool and development server
- **Chakra UI**: Component library with design system
- **React Router**: Client-side routing and navigation

### Data Flow
- **Mock Data**: Currently uses mock data for demonstration
- **State Management**: React hooks for local component state
- **Routing**: React Router for navigation between views
- **Responsive Design**: Breakpoint-based layout adaptation

## 🛠️ Installation & Setup

### Prerequisites
- Node.js 18+ or Bun (recommended)
- Git for version control

### Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/MohaleDesigns/MinitMoney_Dashboard.git
   cd minitmoney
   ```

2. **Install dependencies**
   ```bash
   # Using Bun 
   bun install
   
   # Or using npm
   npm install
   ```

3. **Start development server**
   ```bash
   # Using Bun
   bun run dev
   
   # Or using npm
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173`
